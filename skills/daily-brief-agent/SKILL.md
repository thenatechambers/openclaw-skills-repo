# Daily Brief Agent

An intelligent morning briefing skill that aggregates information from multiple sources and delivers a concise, actionable daily summary.

## What It Does

Every morning (configurable), this skill:
1. **Checks your calendar** for today's meetings and prep needs
2. **Scans priority emails** for urgent items requiring attention
3. **Curates news** from your RSS feeds or specified sources
4. **Reviews action items** from yesterday's notes/tasks
5. **Generates a brief** with AI-powered insights and recommendations
6. **Delivers via** Slack, email, or console output

## Use Cases

- **Founders/Execs:** Start the day informed without app-switching
- **Remote workers:** Async daily standup replacement
- **Knowledge workers:** Context gathering before deep work
- **Teams:** Shared brief with relevant cross-functional updates

```bash
# Via ClawHub
claw install daily-brief-agent

# Via GitHub
claw install https://github.com/thenatechambers/openclaw-skills-repo/tree/main/skills/daily-brief-agent
```

## Configuration

Create `~/.openclaw/skills/daily-brief-agent/config.json`:

```json
{
  "schedule": "0 8 * * 1-5",
  "timezone": "America/New_York",
  "sources": {
    "calendar": {
      "enabled": true,
      "provider": "google",
      "lookahead_hours": 12
    },
    "email": {
      "enabled": true,
      "provider": "gmail",
      "max_items": 10,
      "filter": "label:important"
    },
    "news": {
      "enabled": true,
      "feeds": [
        "https://hnrss.org/frontpage",
        "https://news.ycombinator.com/rss"
      ],
      "max_items": 5
    },
    "tasks": {
      "enabled": true,
      "source": "linear",
      "include_overdue": true
    }
  },
  "delivery": {
    "channel": "slack",
    "slack_webhook": "${SLACK_WEBHOOK_URL}",
    "format": "markdown"
  },
  "ai": {
    "model": "claude-3-5-sonnet",
    "max_tokens": 2000,
    "style": "concise_bullets"
  }
}
```

## Usage

### Manual Run

### Scheduled (Cron)
Add to your OpenClaw crontab:
```bash
0 8 * * 1-5 claw run daily-brief-agent
```

## Customization

### Adding Custom Data Sources

Edit `scripts/custom-source.mjs`:

```javascript
export async function fetchCustomData(config) {
  // Your custom data fetch logic
  const data = await fetchYourApi(config.apiKey);
  return {
    section: "Custom Metrics",
    items: data.metrics.map(m => ({
      title: m.name,
      value: m.value,
      trend: m.trend
    }))
  };
}
```

### Changing Brief Style

Modify the AI prompt in `config.json`:

```json
{
  "ai": {
    "style": "executive_summary",
    "prompt_addendum": "Focus on revenue-impacting items and team blockers."
  }
}
```

## How It Works

1. **Aggregation Phase** — Pulls raw data from all enabled sources
2. **Filtering Phase** — Applies relevance scoring and deduplication
3. **Enrichment Phase** — AI adds context, prep suggestions, insights
4. **Formatting Phase** — Structures output per your format preference
5. **Delivery Phase** — Sends via configured channel(s)

## Troubleshooting

**Brief not generating:**
- Check `~/.openclaw/logs/daily-brief-agent.log`
- Verify all API credentials are valid
- Test individual sources: `claw run daily-brief-agent --test-source=calendar`

**Missing calendar events:**
- Ensure Google Calendar API is enabled
- Check that credential path is absolute

**News not appearing:**
- Verify RSS feeds are valid
- Some feeds require User-Agent header (configured by default)
